# MediDesign Lab

사진을 업로드하면 AI가 맞춤형 의료 액세서리 디자인 시안을 생성해주는 웹앱.
보청기 케이스, 인솔, 손목·무릎 보호대를 심미성과 착용감 모두 고려해 제안합니다.

---

## 현재 상태

**MVP 1단계 진행 중** — 기본 기능 구현 완료, 실사용 테스트 단계

- 🟢 카테고리 선택 → 사진 업로드 → AI 디자인 시안 생성
- 🟢 마음에 드는 시안 저장 및 모아보기
- 🟡 이미지 생성 기능 (2단계 예정)
- ⚪ 3D/CAD 연동 (3단계 이후)
