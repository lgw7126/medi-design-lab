
# MediDesign Lab

## 프로젝트 개요
사용자가 신체 부위 사진을 업로드하면 AI가 맞춤형 의료 액세서리
(보청기 케이스, 인솔, 손목/무릎 보호대) 디자인 시안을 생성하는 웹앱.
심미성과 착용감을 모두 고려한 "의료 + 디자인" 결합 컨셉.

## 운영자 배경
- 비개발자, 디자인/금속공예 배경 (라이노 쥬얼리 CAD 경력)
- 모든 코드는 Claude Code가 작성, 운영자는 기획/디자인 방향 담당
- 설명은 쉽게, 전문용어 최소화

## MVP 범위 (1단계)
- 카테고리 선택 → 사진 업로드 → AI 디자인 시안 생성 → 저장
- 실제 3D 모델/CAD 파일 생성은 하지 않음 (2단계 이후)
- 모바일 우선 반응형 웹

## 기술 스택
- 프론트: React (간단하게)
- AI: Anthropic API (디자인 시안 텍스트 생성)
- 배포: 추후 결정 (Vercel 등)

## 작업 규칙
- 한 번에 하나의 기능만 작업
- 변경 전 반드시 무엇을 할지 한국어로 먼저 설명
- 에러 발생 시 ERRORS.md에 기록
- 진행 상황은 MEMORY.md에 기록

## 디자인 방향
- 시니어도 쓸 수 있게 큰 글씨, 단순한 흐름
- 의료기기 느낌보다 라이프스타일 제품 느낌
